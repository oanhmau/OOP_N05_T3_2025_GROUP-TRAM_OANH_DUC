# Thuộc tính
1. Khách hàng (KháchHàng)
•	Thuộc tính:
o	Tên khách hàng
o	Mã khách hàng
o	Số điện thoại
•	Hành vi (Method):

o	Xem lịch sử nạp tiên
2. Mã máy 
•	Thuộc tính:
o	Tên máy
o	Mã máy
o	Giá chơi
o	Số giờ chơi
o	Số giờ còn lại
•	Hành vi (Method):
o	Cập nhật giờ chơi
3. Thanh toán
•	Thuộc tính:
o	Mã giao dịch
o	Tên khách hàng
o	Số giờ chơi
o	Thành tiền

o	Tính tổng tiền giao dịch
o	Xuất hóa đơn
